Skibidi Video

Project Info:

A huge building maze full of Skibidi videos, good experience to walk and watch 3D videos, see Skibidi walking beside you, collect all coins to watch more and more videos, each video you watch will cost you 1 coin.

Project Details:

  • Company: Gamox
  • Position: Unity Game Developer
  • Tasks:
    - Create a 3D watching videos experience
    - Plan and implement game functionality
    - Create the game UI
    - Create building 3D models and characters animation
    - Implement game effects and sounds
  • Technologies:Unity 3D, C#, Blender, Photoshop
  • Date:January 2024
